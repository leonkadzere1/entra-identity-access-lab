# Microsoft Entra Identity & Access Lab

## Overview
This project documents hands-on labs completed as part of the Microsoft Entra Identity and Access learning path. The lab focuses on identity management, access control, and security features within Microsoft Entra.

## Objectives
- Create and manage users
- Configure groups
- Implement password protection
- Configure Self-Service Password Reset (SSPR)
- Enable Multi-Factor Authentication (MFA)
- Configure Conditional Access policies

## Technologies Used
- Microsoft Entra ID
- Microsoft 365 Admin Center
- Azure Portal
- Conditional Access
- Multi-Factor Authentication

## Lab Modules
- User Management
- Group Management
- Password Protection
- Self-Service Password Reset (SSPR)
- Multi-Factor Authentication (MFA)
- Conditional Access

## Skills Demonstrated
- Identity management
- Access control
- Security configuration
- Role-based access control
- Conditional access configuration
